# Rod's Fangame Tools

This mod adds some cool <cg>utilities</c> for anyone who wishes to make <cy>Geometry Dash fangames</c> or just wants to mess around with level data.

## Features

- Level Utilities
  - Copy Main Levels
  - Copy Level String
  - Paste Level String
  - Encoded/Decoded Level Type
- Miscellaneous
  - Preview Main Level Data


## Recommended Mods

**Unlisted Objects in Editor**: This mod adds useful objects in the editor (most importantly, the secret coin).

<mod:legowiifun.unlisted_objects_in_editor>

**More Icons**: This mod lets you create and test new icons which you can add to your fangames.

<mod:hiimjustin000.more_icons>

**Stats Display API**: This mod (co-developed by yours truly) is an API that lets you add custom stats to the icon kit area. This may be useful if your fangame will include custom stats.

<mod:capeling.garage-stats-menu>

**GDShare**: This mod allows you to share levels with others. This mod can also fix a known bug (by RobTop) where secret coins get automatically deleted from the level each time it's reopened (although you would need to backup the level each time).

<mod:hjfod.gdshare>

**Copy Main Levels**: This mod lets you copy the main levels, in case you have issues with copying the main levels using this mod.

<mod:dominodev.copymainlevels>

## Incompatibilities

- Eclipse Menu: Read-only level preview code may overlap a bit, but it seems to work fine.
